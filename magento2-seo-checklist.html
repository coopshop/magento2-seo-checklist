Group Item General Advice Action for Magento 2 Developers
URL Descriptive URLs: Use a descriptive page url, which should reflect your targeted keyword Enable "Use Categories Path for Product URLs" under Stores => Configuration => Catalog => Search Engine Optimization
Subdomain or subfolder Subdomains are seen as separate domains Choose whichever is easier for you
Hyphens Split words using hyphens Use hyphens as the separator in the URL keys of CMS pages
www or no-www Provide both domains, but set a prefered version in Google Webmaster Tools Use the prefered domain in base URLs
Localisation: Choose a country-specific domain, for better local search results Use a country-specific domains for each website, store or store view targeting the specific country
HTTPS Security is a top priority for Google Implement full site HTTPS to your store
URL builder Use this tool to add custom compaign parameters to your URLs Use the provided tool to generate URLs in your custom web-based or email ad campaigns for your store
File extension Do not strip out the file extension on URLs Append a suffix ".html" to the URL keys of CMS pages
Accessibility 403 Errors Provide a 403 - Access denied page Provide a custom design for 503 error page under pub/errors and optionally Provide a user friendly 403 page with Apache / Nginx
404 Errors Provide a 404 - Page not found page Provide a custom design for error pages under pub/errors and the 404 page under CMS pages
Robots Block pages which should not be indexed via the robots.txt file or <meta name="robots" content=""> Go to Stores => Configuration => General => Design => Search Engine Robots to edit the robots.txt file
File not found Avoid 404 FILE_NOT_FOUND errors Fix any 404 errors before site launching and monitor for 404 errors after going live
Redirects Avoid redirects if possible. Use 301 redirect instead of 302 Manage URL rewrites/rediretions under Marketing => SEO & Search => URL Rewrites
Pagination Implement the rel="next" and rel="prev" attributes to links Implement the rel="next" and rel="prev" attributes to the "Next Page" and "Prev Page" buttons in toolbar, especially for category pages
Moving a website Redirect all your links to the new location via .htaccess Redirect all your links to the new location via .htaccess or "URL Rewrites"
Performance Performance and loading time is important Optimize your store performance as much as possible within the budget, especialy for mobile pages.
Web Performance Optimization is a big topic by itself, see https://github.com/davidsonfellipe/awesome-wpo
Validation Write valid code Use HTML Validator (https://validator.w3.org/) and CSS Validator (https://jigsaw.w3.org/css-validator/) to validate your pages
WAI-ARIA Use WAI-Aria tags to help machines understanding your code Apply ARIA optionally to improve your store accessbility
RichSnippets Markup your code with rich snippets, they show up on the search results page This feature is enabled in Magento 2 by default.
Custom Search With sitelink Google Sitelink search box, people can reach your content more quickly Setup the sitelinks search box by adding specific markup to your homepage
Layout Use divs instead of tables for layout. Using tables is semantically not correct. Use sematic markup when building the Magento theme
Meta Information Title Each page should have a unique speaking title (60 - 100 characters) <title>Website Title</title> Set "Mask for Meta Title" in Stores => Configuration => Ctatlog => Product Fields Auto-Generation
Description Each page should have a unique description (max. 160 characters) <meta name="description" content=""> Set "Mask for Meta Description" in Stores => Configuration => Ctatlog => Product Fields Auto-Generation
Keywords Single Every page should have a single unique targeted keyword Review this with client or the SEO personnel
Research Rank for keywords with high traffic and less competition Use Google Trends
URL Keyword should appear in URL name Review this with client or the SEO personnel
Title Keyword should appear in page title Review this with client or the SEO personnel
Heading Keyword should appear in headings Review this with client or the SEO personnel
Content Keyword should apear in ~3% of article length Review this with client or the SEO personnel
Meta Tag You can ommit the <meta name="keywords" content="">, search engines do not use this meta tag Unset "Mask for Meta Keywords" in Stores => Configuration => Ctatlog => Product Fields Auto-Generation
Content Content Content matters the most in SEO Review this with client or the SEO personnel
Headings Clear structure H1 -H6 max. 70 characters long Don't mark the site logo as H1 if it is not homepage
Strong Use <strong> tag to highlight your targeted keyword Review this with client or the SEO personnel
Uniqueness Do not provide duplicated content, use unique content types Enable "Use Canonical Link Meta Tag For Categories/Products" under Stores => Configuration => Catalog => Search Engine Optimization
Length Article should be at least 300 words Check it for your store blog posts
Freshness New content is important. Updating pages or regulary posting is recommended Add a store blog along with your online store
Flash Avoid Flash content and flash pages. They are not accessible on mobile phones and will be ranked lower Avoid using Flash, use CSS transition and animiation, SVG instead
Images File name Use a short descriptive name Use hyphens as the separator in the image names
Alt attribute Add an alt-attribute this a description of the image (60 - 70 characters) Always add the "alt" attribute to <img> tags
Responsive Images Serve the most optimized image corresponding to the window size Use the <picture> tag or the srcset and sizes attributes in <img> tag when they are appropriate
Size Keep the filesize as low as possible Integrate gulp-imagemin or grunt-contrib-imagemin into your frontend workflow
Optimization Optimize images by removing some meta information Integrate gulp-imagemin or grunt-contrib-imagemin into your frontend workflow
Videos Controls Add controls to playback and control your video Add the "controls" attribute to <video> tags
Transcriptions Use transcriptions for indexing, usability & content Use the <track> tag if subtitles are available
Unplayable content Avoid unplayable video content. Use HTML5 <video> tag instead of Flash Avoid using Flash, use <video> tag instead
Links Title Add the title attribute to links Alwasy add the "title" attribute when building the storefront
Backlinks Only add external links if you got a backlink to your site Review this with client or the SEO personnel
nofollow Add rel="nofollow" attribute to external links only to prevent spam and bad links Review this with client or the SEO personnel
Internal links Add ~3 internal links to your content Review this with client or the SEO personnel
Languages The hreflang tag tells Google which language you are using on a specific page, so the search engine can serve that result to users searching in that language <link rel="alternate" href="example.com/fr/" hreflang="fr-fr" /> Add "herflang" attributes in sitemap or your web pages when there are multiple stores
Naming Use a descriptive link name: “Click here” or “Read more” are bad link text. Better “Read more about SEO and Web Accessibility” Review this with designers
Mobile Viewport Tell browsers how to adjust the page's dimension and scaling to suit the device <meta name="viewport" content="width=device-width, initial-scale=1"> Do it in your page template
mobile friendly Mobile optimized sites are marked in search results. Test for mobile friendly site
AppLinks Apps that link to your content can then use this metadata to deep-link into your app Apply this only when you have mobile apps for the store
Tap targets Clickable links should not be too small Tweak the style of links and buttons on mobile to be big enough to tap easily
Smart App Banner Safari has a Smart App Banner feature that provides a standardized method of promoting apps on the App Store from a website. Apply this only when you have mobile apps for the store
Sitemap HTML sitemap An HTML sitemap allows site visitors to easily navigate a website. Magento provide HTML sitemap by default, just make sure the sitemap link is not removed in storefont and improve the sitemap redering performance when you have large catalog
XML sitemap Help search engines to index your pages Congiure XML sitemaps under "Marketing => SEO & Search => Site Map" and "Stores => Configuration => Catalog => XML Sitemap"
Image sitemap Increase that your images can be found in Image Search results Optional
Video sitemap Make sure, search engines know about all the video content on your site Optional
Mobile sitemap For feature phones, you can create a mobile sitemap Only implement this when your target audience using feature phones
Social Media Social Shares Provide sharing options for your site Optional
Social Profiles Add social profiles to your Google search results Implement this when your business has relavent sociall network accounts
Facebook Sharing Best Practices for Websites & Mobile Apps Implement this when you want to share your product to Facebook
Twitter With Twitter cards, you can attach photos, videos and media experience to you Tweets Implement this when you want to share your product to Facebook
Google+ Authorship Add Google+ authorship information to your page <link rel="author" href="//plus.google.com/u/0/[GOOGLE+ ID]"> or<a href="//plus.google.com/u/0/[GOOGLE+ ID]" rel=author">Google+ Profile</a> or <a href="//plus.google.com/u/0/[GOOGLE+ ID]" rel=me">Google+ Profile</a> or <a href="//plus.google.com/u/0/[GOOGLE+ ID]" rel=publisher">Google+ Profile</a> Implement it when you have a blog along with your store
Note: the gray items mean they involve content manager or SEO personnel
